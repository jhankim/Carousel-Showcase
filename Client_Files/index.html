Debug Logging


In order to turn on debug logging for the JavaScript console add ?debug=true to the URL string



Automatically Generated Carousel



Space to Generate Carousel from Command Line


In order to create the new carousel enter "var test2 = new olapicCarousel('#carousel2');" into your javascript console and it will initialize the new carousel with the pre-existing div on the page. You can also add two optional integers after the div ID to set the minmum amount of slides and the maximum amount of slides for the carousel.
